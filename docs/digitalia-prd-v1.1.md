# 1. 製品概要

## 1.1 プロダクト名
**DIGITALIA（デジタリア）** - The Code Journey through 520 Programming Terms

## 1.2 コンセプト
プログラミング初心者が、ファンタジー世界「デジタリア」を冒険しながら、520のプログラミング用語を**ストーリー体験型学習**で楽しく学習できるWebアプリケーション。

**コア体験**：テレプロンプター形式で流れるストーリーを読み（または聴き）ながら、登場する用語をその場で理解し、自分の「マイフォルダ」にコレクションしていく。

## 1.3 ターゲットユーザー
- プログラミング初心者
- Web開発を学び始めた人
- 用語の意味を効率的に覚えたい人
- ゲーミフィケーションのある学習を好む人
- 読書感覚で学習したい人

## 1.4 プロダクトビジョン
> 「すべての旅人に、コードの翼を」

520匹のコードバタフライを集める旅を通じて、プログラミングの言葉を自然と身につけ、自信を持ってコードを書き始められるようになる。

## 1.5 学習モード

| モード | 説明 | 特徴 |
|--------|------|------|
| **ストーリーモード** | テレプロンプター形式でストーリーを体験 | メイン体験、没入感重視 |
| **フラッシュカードモード** | 従来のカード学習 | 復習・反復学習向け |
| **検索・辞書モード** | 用語を直接検索・参照 | リファレンスとして使用 |

# 2. 世界観・ストーリー

## 2.1 舞台設定
- **世界名**: デジタリア（DIGITALIA）
- **コンセプト**: 0と1で構成されたデジタル世界
- **雰囲気**: レトロフューチャー × ファンタジー × 温かみのある光

## 2.2 ストーリー概要
主人公（学習者）は、デジタリアの世界に迷い込み、ガイドキャラクターと共に16の地域を冒険。各地域で用語を学ぶたびに「コードバタフライ」が誕生し、500匹集めると「メモリーガーデン」に到達。エピローグで最後の20語を学び、520匹の蝶と共に現実世界に帰還する。

## 2.3 メインキャラクター

### 2.3.1 ビットくん（Bit-kun）
- **役割**: メインガイド（全編を通じて登場）
- **見た目**: メタリックな球体ロボット、レンズのような大きな目、赤いLEDライト、周囲に浮遊する小さな球体
- **性格**: 好奇心旺盛、明るく元気、初心者に優しい
- **口調**: 「〜だよ」「〜なんだ」

#### 登場タイミング・表情設定

| 場面 | 表情 | セリフ例 | 出現位置 |
|------|------|---------|---------|
| 章開始 | excited | 「新しい冒険の始まりだよ！」 | 画面中央→右へ移動 |
| 用語登場 | normal | 「これが〇〇だよ」 | カードの横 |
| 用語説明中 | thinking | 「つまりね...」 | カードの横（首かしげ） |
| 正解・理解時 | happy | 「すごい！覚えたね！」 | 跳ねるアニメーション |
| 不正解・未理解時 | encourage | 「大丈夫、もう一度見てみよう」 | 寄り添うように近づく |
| 章クリア | proud | 「やったね！次の冒険へ！」 | 画面中央で祝福 |
| 休憩提案（30分経過） | sleepy | 「少し休憩しようか...」 | ゆっくり浮遊 |
| エラー発生時 | surprised | 「あれ？何か起きたみたい」 | 点滅 |

### 2.3.2 バイトさん（Byte-san）
- **役割**: ビットくんの先輩（重要な場面で登場）
- **見た目**: 8つの光が集まった、少し大きな存在
- **性格**: 穏やか、頼りになる、まとめ役
- **口調**: 「〜です」「〜ですね」

#### 登場タイミング

| 場面 | 役割 | セリフ例 |

## 3.3 シーン構造

### 3.3.1 シーンデータ構造

```typescript
interface Scene {
  id: string;                      // シーンID
  chapterId: number;               // 章番号
  order: number;                   // シーン順番
  title: string;                   // シーンタイトル
  content: SceneContent[];         // コンテンツ配列
  characters: CharacterAppearance[]; // 登場キャラクター
  relatedTerms: string[];          // このシーンで登場する用語ID
  duration: number;                // 推定所要時間（秒）
  bgm?: string;                    // BGMファイル
  background?: string;             // 背景画像
}

interface SceneContent {
  type: 'narration' | 'dialogue' | 'term_highlight' | 'code_block';
  speaker?: string;                // 話者（dialogueの場合）
  text: string;                    // テキスト内容
  termId?: string;                 // ハイライトする用語ID
  emotion?: string;                // キャラクターの表情
}

interface CharacterAppearance {
  characterId: string;             // キャラクターID
  position: 'left' | 'center' | 'right';
  initialEmotion: string;          // 初期表情
  enterAnimation: string;          // 登場アニメーション
  exitAnimation?: string;          // 退場アニメーション
}
```

## 3.4 用語カード連携

### 3.4.1 用語ハイライト表示

ストーリー内で用語が登場すると：

1. **視覚的ハイライト**
   - 用語が金色に光る
   - 下線が引かれる
   - 軽いパルスアニメーション

2. **インタラクション**
   - クリック/タップで用語カードがポップアップ
   - ストーリーは自動で一時停止

3. **カードポップアップ**
