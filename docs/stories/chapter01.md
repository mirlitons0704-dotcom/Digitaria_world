# 第1章：コンピュータの世界へ

**〜 0と1の冒険のはじまり 〜**

**カテゴリ：基礎概念（Basics）**
**登場用語：Binary, Bit, Byte, ASCII, Machine language, Code, Program, App, Software, File**

---

## シーン4：ビットの仲間たち

```
歩いていると、ビットくんそっくりの存在が
7人並んでいるのが見えた。

「あ、みんな！紹介するね。
ぼくたちビットは、8人集まると『バイト』になるんだ」

8人のビットが横一列に並ぶと、
ふわっと光り輝いて、ひとつの大きな存在になった。

「こんにちは。私はバイトです」

穏やかな声だった。

「バイトさんになると、もっとたくさんのことが
表現できるようになるんだ。
たとえば、1バイトで256種類もの違いを表せるんだよ！」

「256種類……！」

「そう。0から255までの数字とか、
アルファベットの『A』から『Z』とか。
8人の力を合わせると、できることが
ぐーんと広がるんだ」
```

**💡 用語登場：Byte（バイト）**

---

## シーン5：文字の秘密

```
バイトさんが、不思議な表を見せてくれた。

「これは ASCII（アスキー）コード表といいます。
私たち1バイトで、どの数字がどの文字を表すか
決めた約束事なんです」

表を見ると、「65」の横に「A」、
「66」の横に「B」と書いてある。

「つまり、コンピュータが『01000001』という
0と1の並びを見たら、『A』という文字だと
わかるってこと？」

「その通りです」

ビットくんが嬉しそうに跳ねた。

「すごいでしょ！ぼくたちの並び方で、
文字になったり、数字になったり、
いろんなものを表現できるんだ！」
```

**💡 用語登場：ASCII（アスキー）**

---

## シーン6：機械のことば

```
少し歩くと、奇妙な看板が立っていた。

「01001000 01000101 01001100 01001100 01001111」

「これ、なんて書いてあるの？」

ビットくんが教えてくれた。

「Machine language——機械語だよ。
コンピュータが直接理解できる唯一の言葉なんだ。
この看板は『HELLO』って書いてあるんだよ」

「0と1だけで『HELLO』……」

「昔の人間は、こうやって0と1を
ずらーっと並べてコンピュータに命令していたんだ。
大変だったろうね……」

「それは……確かに大変そう」

「だから人間は、もっとわかりやすい方法を
考えたんだ。それが『コード』だよ」
```

**💡 用語登場：Machine language（機械語）**

---

## シーン7：人間の言葉に近づける

```
次の場所には、もう少し読みやすい文字が並んでいた。

print("HELLO")

「これは Code（コード）——
プログラミング言語で書かれた命令文だよ」

「あ、これなら少し読める気がする！」

「でしょ？人間が読み書きしやすいように
作られた言葉なんだ。
これをコンピュータが理解できる機械語に
変換してくれる仕組みがあるんだよ」

「通訳みたいなものかな？」

「そうそう！まさに通訳！
人間の言葉と機械語の橋渡しをしてくれるんだ」
```

**💡 用語登場：Code（コード）**

---

## シーン8：命令の束

```
コードが書かれた紙が何枚も積み重なっている場所に来た。

「たくさんのコードが集まると、
Program（プログラム）になるんだ」

「プログラム……運動会のプログラムみたいなもの？」

「いい例えだね！
運動会のプログラムには『開会式→徒競走→玉入れ→……』
って順番が書いてあるでしょ。
コンピュータのプログラムも同じで、
『これをやって→次にこれをやって→……』
という命令の順番が書いてあるんだ」

「なるほど、やることリストか！」

「そう。プログラムは、
コンピュータへの『やることリスト』なんだよ」
```

**💡 用語登場：Program（プログラム）**

---

## シーン9：便利な道具たち

```
街に着くと、いろいろなお店が並んでいた。
「電卓店」「メモ帳店」「地図店」「カメラ店」……

「これはみんな App（アプリ）だよ。
Application の略で、特定の目的のために
作られたプログラムのことなんだ」

「スマホに入ってるアプリと同じ？」

「まさにそれ！
電卓アプリは計算するため、
カメラアプリは写真を撮るため。
それぞれ決まった目的があるプログラムを
アプリって呼ぶんだ」
```

**💡 用語登場：App（アプリ）**

---

## シーン10：ソフトとハード

```
「ところで、アプリとソフトウェアって違うの？」

「いい質問だね！
Software（ソフトウェア）は、
コンピュータ上で動くプログラム全般のことだよ。
アプリはソフトウェアの一種なんだ」

ビットくんが図を描いてくれた。

「きみの世界で言うと……
『ソフトウェア』は『料理』全般、
『アプリ』は『パスタ』や『カレー』みたいな
具体的なメニューかな」

「ソフトウェアの方が大きな概念ってこと？」

「そうそう！
そして、料理を作る『キッチン』や『フライパン』が
Hardware（ハードウェア）——
パソコン本体とかスマホ本体のことだよ」
```

**💡 用語登場：Software（ソフトウェア）**

---

## シーン11：データの入れ物

```
最後に、図書館のような建物に案内された。
棚にはたくさんの箱が並んでいる。

「これは File（ファイル）だよ。
データを保存しておくための入れ物なんだ」

「書類を入れるファイルと同じイメージ？」

「ぴったり！
文章を入れたファイル、写真を入れたファイル、
音楽を入れたファイル……
種類ごとに整理して保存しておけるんだ」

箱をひとつ開けてみると、
中にはきれいに整理されたデータが入っていた。

「次の章では、このファイルの種類について
もっと詳しく教えるね。
いろんな拡張子があるんだよ！」

「拡張子？」

「ふふ、お楽しみに！」
```

**💡 用語登場：File（ファイル）**
