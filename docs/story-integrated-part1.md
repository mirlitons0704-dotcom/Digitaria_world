# 世界観設定

## 舞台：「デジタリア」

私たちが毎日使うコンピュータやスマートフォン。
その画面の向こう側には、「デジタリア」と呼ばれる広大な世界が広がっている。

デジタリアは16の地域に分かれており、それぞれの地域には独自の文化と技術がある。
この世界の住人たちは、「コード」という言葉を使って様々なものを創り出す。

## 主人公：あなた（プレイヤー）

ある日、パソコンの画面に吸い込まれるようにしてデジタリアに迷い込んだ。
元の世界に戻る方法を探しながら、この世界の仕組みを学んでいく。

## 案内役：ビットくん

デジタリアで最初に出会う小さな存在。
0と1の光を纏った、好奇心旺盛なガイド。
難しいことも身近な例えでわかりやすく教えてくれる。

## コードバタフライ 🦋

デジタリアに生息する、光の蝶。
0と1の粒子でできており、羽ばたくたびにキラキラと輝く。
知識の記憶を運び、学習者の成長を見届ける神秘的な存在。

## シーン1：目覚め

```
気がつくと、あなたは見知らぬ場所に立っていた。

周りを見渡すと、すべてが「0」と「1」でできているようだ。
空を流れる雲も、足元の地面も、遠くに見える山々も。

「ここは……どこ？」

すると、目の前にぽっと小さな光が現れた。
```

**💡 用語登場：Binary（2進数）**

## シーン2：ビットくんとの出会い

```
「やあ！目が覚めたんだね！」

光が声を出した。よく見ると、0と1が交互に点滅している
小さな存在だった。

「ぼくはビット。この世界『デジタリア』のガイドだよ」

「デジタリア……？」

「そう！きみのいた世界では『コンピュータの中』って
呼ばれている場所さ。ここではすべてが Binary——
つまり0と1だけで表現されているんだ」

「0と1だけで、こんなにいろんなものが？」

「信じられないよね。でも本当なんだ。
たとえば、きみの世界の『電気のスイッチ』を思い出して。
ON か OFF か、2つの状態しかないでしょ？
コンピュータも同じ。電気が流れているか、いないか。
それを0と1で表しているんだよ」
```

**💡 用語登場：Bit（ビット）**

## シーン3：コードバタフライとの出会い 🦋

```
ビットくんと歩き始めると、ふわりと光が舞った。

「あ……蝶？」

見上げると、光り輝く蝶が何匹か飛んでいる。
よく見ると、羽は0と1の粒子でできていて、
羽ばたくたびにキラキラと輝いていた。

「きれい……」

「あれは『コードバタフライ』」
ビットくんが嬉しそうに言った。

「この世界に住む、とても大切な存在なんだ。
コードバタフライは、知識の記憶を運んでいるんだよ」

「知識の記憶を……運ぶ？」

「そう。デジタリアには、たくさんの『用語』という花が
咲いているんだ。コードバタフライは、
花から花へと飛び回って、意味を受粉させるように広げていく。
おかげで、この世界の知識はバラバラじゃなくて、
ちゃんと繋がっているんだ」

一匹の蝶が、あなたの指先にそっと止まった。
羽が触れた場所が、ほんのり温かい。

「わ……」

「気に入られたみたいだね」
ビットくんが微笑んだ。

「そしてね、もうひとつ大切なことがある。
きみが用語を本当に理解したとき——
ただ覚えただけじゃなくて、心から『わかった！』と思えたとき——
その知識は光の粒になって、新しい蝶が生まれるんだ」

「ぼくが学ぶと、蝶が……？」

「うん。最初は小さな光の粒。
でも学びが深まるにつれて、それは卵になり、
幼虫になり、さなぎになり……
そしていつか、こうして美しい蝶になって羽ばたく」

ビットくんは遠くを指さした。
丘の向こうに、無数の光が舞っているのが見える。

「あそこは『メモリーガーデン』。
500の言葉を学び終えた人だけが行ける、特別な場所だよ」

「500……」

「この旅で学ぶ用語の数さ。
全部の言葉を理解したとき、きみの蝶たちが
あの丘を舞っているはずだよ」

光の蝶が指先から飛び立ち、仲間と一緒に空へ消えていった。

あなたは心に決めた。
いつか、あの丘で自分の蝶たちに会いに行こう、と。
```

## シーン4：ビットの仲間たち

```
歩いていると、ビットくんそっくりの存在が
7人並んでいるのが見えた。

「あ、みんな！紹介するね。
ぼくたちビットは、8人集まると『バイト』になるんだ」

8人のビットが横一列に並ぶと、
ふわっと光り輝いて、ひとつの大きな存在になった。

「こんにちは。私はバイトです」

穏やかな声だった。

「バイトさんになると、もっとたくさんのことが
表現できるようになるんだ。
たとえば、1バイトで256種類もの違いを表せるんだよ！」

「256種類……！」

「そう。0から255までの数字とか、
アルファベットの『A』から『Z』とか。
8人の力を合わせると、できることが
ぐーんと広がるんだ」
```

## シーン5：文字の秘密

```
バイトさんが、不思議な表を見せてくれた。

ビットくんが嬉しそうに跳ねた。

「すごいでしょ！ぼくたちの並び方で、
文字になったり、数字になったり、

```

```

大変だったろうね……」

「それは……確かに大変そう」

「だから人間は、もっとわかりやすい方法を
考えたんだ。それが『コード』だよ」

## シーン8：命令の束

```
コードが書かれた紙が何枚も積み重なっている場所に来た。

「たくさんのコードが集まると、
Program（プログラム）になるんだ」

「プログラム……運動会のプログラムみたいなもの？」

「いい例えだね！
運動会のプログラムには『開会式→徒競走→玉入れ→……』
って順番が書いてあるでしょ。
コンピュータのプログラムも同じで、
『これをやって→次にこれをやって→……』

「なるほど、やることリストか！」

「そう。プログラムは、
コンピュータへの『やることリスト』なんだよ」

## シーン8：カスタムフック

```
「Custom hook（カスタムフック）は、
自分で作るオリジナルのフックよ」

function useCounter(initial = 0) {
  const [count, setCount] = useState(initial);
  const increment = () => setCount(c => c + 1);
  const decrement = () => setCount(c => c - 1);
  return { count, increment, decrement };
}

// 使い方
const { count, increment } = useCounter(10);

「ロジックを再利用できるのが
カスタムフックの魅力よ」

「フック名は必ず use で始めること。
これが React のルールよ」
```

**💡 用語登場：Custom hook**
